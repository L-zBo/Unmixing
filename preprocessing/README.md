# Preprocessing 预处理模块

本目录包含拉曼光谱预处理的完整流程，主线协议为 `ALS 基线校正 + L2 归一化`。

## 📁 文件列表

| 文件名 | 功能 | 状态 |
|--------|------|------|
| `preprocess.py` | 完整预处理流程（ALS / L2 / max / none 多协议可切换） | 🚧 C4 搬入 |
| `endmembers.py` | 端元谱加载与端元矩阵构建（PE / PP / 多源淀粉） | 🚧 C4 搬入 |
| `preprocess_dataset.py` | 命令行入口：把 `dataset/` 整体跑一遍预处理 | 🚧 C4 搬入 |
| `README.md` | 本文档 | ✅ |

---

## 🎯 核心功能

### 1. ALS 基线校正
去除拉曼光谱的荧光背景与基线漂移。

### 2. L2 归一化
统一光谱强度量纲，消除样本间差异。

### 3. 协议化预处理
支持以下三种预处理协议：
- `als_l2`（主线）
- `als_max`
- `none_l2`

### 4. 端元谱库管理
支持多种淀粉来源端元：
- 主线：`玉米淀粉`
- 泛化：`展艺玉米淀粉` / `新良小麦淀粉` / `甘汁园小麦淀粉`

---

## 📊 与主线的关系

```
原始 dataset/
    ↓
preprocess.py (ALS+L2)
    ↓
预处理后的光谱
    ↓
endmembers.py 构建端元矩阵
    ↓
unmixing/unmix.py 跑 NNLS 解混
```
